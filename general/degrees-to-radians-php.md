---
title: "Degrees to Radians"
description: "Convert degrees to radians"
topic: "math"
parameters:
  - name: "degrees"
    type: "number"
    default: 180
    description: "Angle in degrees"
exampleOutputs:
  - input: { "degrees": 180 }
    output: "3.141592653589793"
  - input: { "degrees": 90 }
    output: "1.5707963267948966"
---

```php
deg2rad($degrees);
```
