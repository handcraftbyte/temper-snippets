---
title: "Reverse Array"
description: "Reverse the order of elements in an array"
topic: "arrays"
parameters:
  - name: "arr"
    type: "array"
    default: [1, 2, 3, 4, 5]
    description: "Array"
exampleOutputs:
  - input: { "arr": [1, 2, 3, 4, 5] }
    output: "[5, 4, 3, 2, 1]"
---

```ruby
arr.reverse
```
